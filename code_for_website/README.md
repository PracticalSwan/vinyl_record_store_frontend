# Design Import Snapshot

This folder preserves the original imported Groovehaus design implementation for reference. It is not the active frontend and is excluded from the repository's primary lint command.

Use the root `src/`, `package.json`, and Vite configuration for all current development, validation, and backend integration. Do not synchronize routine feature changes into this snapshot.

The snapshot remains tracked to avoid deleting user-authored design work and to preserve design provenance.
