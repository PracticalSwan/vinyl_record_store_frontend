# Design Reference Policy

This project may look at similar websites for design research, but it must not copy protected work.

## Do Not Copy

Do not copy proprietary:

- Design systems.
- HTML.
- CSS.
- Images.
- Logos.
- Product data.
- Branding.
- Layouts pixel-for-pixel.

## Allowed Use

Similar websites may be used only for research and inspiration. Record observations as general patterns, such as:

- Common navigation patterns.
- Product card information hierarchy.
- Filter types.
- Accessibility patterns.
- Recommendation display patterns.

Do not store copied website code or assets in this repository.

## Research Notes

Prefer manual design analysis notes and approved screenshots over scraping. Keep design references separate from implementation.

If design research is stored, use a separate document or the planned `designReferences` collection. Include source name, source URL, general observations, and usage limits.

## Scraping Policy

Scraping is not approved in the current phase.

If scraping is later approved:

- Scrape only allowed public metadata.
- Respect robots.txt, terms of service, copyright, and rate limits.
- Do not bypass protections.
- Do not scrape private data.
- Do not copy proprietary content into the app.

## Future Updates

Future agents must update this file if the design research policy changes.
